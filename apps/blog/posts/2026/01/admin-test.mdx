# Admin 전용 포스팅 테스트

이 포스팅은 `adminOnly: true` 설정으로 인해 관리자만 접근할 수 있습니다.

## 테스트 항목

1. ✅ 일반 사용자는 목록에서 이 포스팅을 볼 수 없습니다
2. ✅ 일반 사용자가 직접 URL로 접근하면 404 페이지가 표시됩니다
3. ✅ 관리자로 로그인하면 목록에서 이 포스팅을 볼 수 있습니다
4. ✅ 관리자로 로그인하면 정상적으로 포스팅 내용을 볼 수 있습니다
5. ✅ 태그 목록에서 'admin', 'test', 'private' 태그 옆에 자물쇠 아이콘이 표시됩니다

## Admin 전용 기능

이 포스팅은 다음과 같은 경우에 유용합니다:

- 내부 문서나 노트
- 아직 공개하기 전인 초안
- 특정 사용자 그룹에게만 공개하고 싶은 내용

## 확인 방법

1. 로그아웃 상태에서 홈페이지를 확인해보세요 - 이 포스팅이 목록에 없어야 합니다
2. 로그아웃 상태에서 `/posts/2026/01/admin-test`로 직접 접근해보세요 - 404가 표시되어야 합니다
3. 관리자로 로그인한 후 홈페이지를 확인해보세요 - 이 포스팅이 목록에 표시되어야 합니다
4. 태그 목록을 확인해보세요 - admin, test, private 태그 옆에 자물쇠 아이콘이 있어야 합니다
